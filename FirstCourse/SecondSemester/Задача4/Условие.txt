19. В файле data.dat задан двумерный массив целых чисел с различным
количеством элементов в каждой строке. Элементы массива задаются через
пробел. Ограничения на количество строк в массиве нет.
Ввести двумерный массив из файла в массив в программе, отведя соответствующим образом память. 
Определить, существует ли в массиве элемент, равный среднему арифметическому элементов массива из строки, содержащей данный элемент. 
Если существует, то уничтожить из массива столбец с найденым элементом.
Если найдено несколько элементов, то использовать только первый из них. Если элемент не найден, то матрица не
изменяется. Вывести в файл data.res полученный двумерный массив.
